# kadai09_db2

# ①課題名
Skill Log（成長ブックマークアプリ）

## ②課題内容（どんな作品か）
- 日々の学習・行動・挑戦を記録し、自分の成長を可視化するアプリです。
- 挑戦内容、カテゴリ、難易度、結果、メモを登録すると、結果に応じてレートが変動します。
- レートに応じてキャラクターが進化し、ゲーム感覚で成長を振り返れるようにしました。

## ③アプリのデプロイURL
https://meplatest.sakura.ne.jp/php02_hw/

## ④アプリのログイン用IDまたはPassword（ある場合）
なし

## ⑤工夫した点・こだわった点
- 単なる記録アプリではなく、成長が見える仕組みとしてレート制を取り入れました。
- 挑戦結果に応じてレートが変動し、卵 → ひよこ → 小鳥 → 鳥 → 鷹 → 不死鳥とキャラクターが進化する仕様にしました。
- フォーム入力がしやすいように、シンプルかつ見やすいUIを意識しました。
- PHPでPOST・INSERT・SELECTの基本処理を実装し、DBとの連携を意識して作成しました。

## ⑥難しかった点・次回トライしたいこと（又は機能）
- ローカル環境（XAMPP）では動作していたものを、さくらサーバーへデプロイする際のDB接続に苦戦しました。
- ローカル環境と本番環境で接続先情報が異なることを、実践を通して理解できました。
- 次回はログの編集・削除機能を追加したり、難易度によってレート変動量を変えるなど、さらにゲーム性を高めたいです。

## ⑦フリー項目（感想、シェアしたいこと等なんでも）
- 今回は初めて本格的にMySQLとPDOを使った課題で、PHP・DB・デプロイまで一連の流れを学ぶことができました。
- レーティングという仕組みを知ったきっかけは、サッカーW杯の優勝予測でした。
- 数字によって強さや成長を可視化できる考え方が面白く、学習にも応用できるのではないかと感じました。
- 学習は積み重ねが見えにくく挫折しやすいですが、小さな挑戦でも成長として可視化できれば、前向きに継続しやすくなるのではないかと考え、このアプリを作成しました。
- ローカルでは動くのに本番環境で動かないという経験も含め、かなり学びの多い課題になりました。
